# AI Sentiment Checker

This is a beginner-friendly AI project built with Python, Gradio, and Hugging Face Transformers.

## What it does

The user enters a sentence, and the AI predicts whether the text sounds positive or negative.

## Tech used

- Python
- Hugging Face Transformers
- Gradio
- GitHub
- Hugging Face Spaces
